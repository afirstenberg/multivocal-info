Multivocal 0.11.2 now available

0.11.2 is a bugfix release, addressing some issues found when running on Node 8.

Version 0.11.1 introduced support for AoG's "Immersive Canvas" feature, which allows an HTML page to be rendered on Assistants with screens (such as Android and the Home Hub).

An example is now available illustrating some basic functionality for the immersive canvas.

Additional changes/fixes since 0.10.0:

  • Fixes for requirements, particularly the sign-in requirement.
  • Additional information when a requirement is not met.
  • Fix pre-processing path for Google's healthcheck ping.
  • Suggestions can now come from either Msg or Suffix.
  • Documentation clarifying how Response Templates work.

(There are no changes from 0.11.0. The version bump is due to npm having a mangled copy of 0.11.0 and no way to fix it.)
