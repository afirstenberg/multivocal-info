Multivocal 0.17.1 now available

The focus of this release is improved support for Dialogflow CX (Dialogflow 3). Consult the platform notes for some details, but to summarize:

  • Added support for the Dialogflow Messenger and generic telephony integrations.
  • Improved locale handling to make it consistent with other platforms.
  • Webhook request verification using the provided Authorization header.

As part of this, there have been a number of changes behind the scenes which should improve other platforms as well.

(Version 0.17.0 is identical, but there were some omissions in documentation which were fixed for 0.17.1.)
