Multivocal 0.13.1 now available

This release finalizes support for AoG's "Interactive Canvas" feature, now that it is generally available. This allows an HTML page to be rendered on Assistants with screens (such as Android and the Home Hub).

An example is now available illustrating some basic functionality for the immersive canvas.

Specifically, this release adds the following over the preliminary support for the Canvas:

  • Use the final attribute names in the output JSON.
  • Add the 
    Setting/Page/IncludeEnvironment
    setting, which allows you to specify the Environment attributes included in all responses.
  • Improved documentation.
