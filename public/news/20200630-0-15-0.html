Multivocal 0.15.0 now available

This release brings support for two new platforms:

  • Actions Builder / Actions SDK for the Google Assistant (Actions on Google v3)
  • Google Chat (formerly Hangouts Chat) under Dialogflow

The Actions Builder/SDK support directly handles platform features such as

  • First Simple / Last Simple prompts
  • Using the "handler name" as the Action name

Both implementations are a bit raw still, and we welcome feedback (and patches!). But it is also pleasing to note that adding both platforms was fairly straightforward, and we look forward to using these lessons to expand to more platforms.
